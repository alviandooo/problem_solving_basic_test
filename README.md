## Problem solving basic test

# Installation 
- clone project dari repository github
- buka terminal dan arahkan ke direktori project
- lalu ketikan perintah berikut dan tekan enter : npm run start (untuk menjalankan jawaban test 1,2,3)

* Anda bisa menjalankan jawaban test secara terpisah dengan mengetikkan perintah berikut di terminal anda:
- npm run basic-test-1 (untuk menjalankan jawaban test 1)
- npm run basic-test-2 (untuk menjalankan jawaban test 2)
- npm run basic-test-3 (untuk menjalankan jawaban test 3)
